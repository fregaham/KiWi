Information: Bausteine einer Aufgaben/Lernsequenz

Bei der Erstellung einer Aufgaben- bzw. Lernsequenz ist eine bestimmte Grundstruktur vorgegeben.


Es stehen Ihnen folgende Bausteine zur Verfuegung:

  • Angabe bzw. Lerninhalt
  • Zusaetzliche Information(en) zur Angabe bzw. Lerninhalt
  • Aufgabenstellung(en)
  • Zusatzinformation(en) zu jeder Aufgabenstellung
  • Loesungsweg

Jeder dieser Bausteine kann durch

  • die Eingabe von Text und Bild in einem einfachen Editor
  • das Hochladen von Praesentationsfolien
  • das Hochladen von Movie(Film)-Dateien

erstellt werden.
